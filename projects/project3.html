Project 3: Hash Table

This is a visualization of a hash table using double hashing. You can add, remove, and fully empty the table, as well as change its length. I also implemented a stepwise add function which can be run at variable speed so that you can see the indicies being checked by the algorithm. Please note that the hash functions I used are scalable and not constrained to/based on a specific table length. As such, the table will fill quickly depending on the value input. Typically, a rehash function would be used in these cases; I wrote such a function into my hashtable but did not implement it to keep the table size limited for aesthetic purposes. Enjoy!
